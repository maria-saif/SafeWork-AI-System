# SafeWork: AI-Driven Workforce Safety Monitoring System

SafeWork is an intelligent, end-to-end safety monitoring platform designed to minimize workplace hazards and automate safety compliance using Computer Vision and Machine Learning.

## 🚀 Key Features
* **Real-Time PPE Detection:** Computer vision models to verify safety gear compliance (helmets, vests).
* **Geofencing & Proximity Alerts:** Immediate alerting when workers enter restricted hazardous zones.
* **Predictive Risk Assessment:** Machine Learning pipelines that analyze environmental and operational data to forecast potential incidents.
* **Interactive Dashboard:** Complete management dashboard with live analytics, automated logs, and Twilio SMS alerting integrations.

## 🛠️ Tech Stack
* **AI & Machine Learning:** Python, Computer Vision (YOLO), Scikit-Learn.
* **Backend & System Integration:** PHP, Python (AI Endpoints).
* **Database & Frontend:** MySQL, HTML5, CSS3, JavaScript.

## 📁 Repository Structure
* `risk_model.py` & `worker_cv.py`: Core AI and predictive analytics pipelines.
* `ai_endpoint.py`: API bridge between Python AI models and the PHP backend.
* `*.php`: Dashboard, telemetry analytics, and SMS notification system.
